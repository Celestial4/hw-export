1.运行前先安装python，把python的安装位置记录下来
2.设置环境变量，pythonpath=xxx，其中xxx为python的安装位置，例如：D:\Program Files\Python310
3.执行setup.py python脚本， 命令： python setup.py
4.配置使用文件config.json
5.开始使用
    使用实例：
        python run.py

6.config.json说明：
    accessKey、secretKey：数据查询身份信息
    queryField：查询依据
    table_name：命令行提示字符串
    table_id：查询表的元数据
